12  * This header is the interface of the platform System Info API.
13  * The user will be able to get the following system information:
14  * - getting the date and time as a string,
15  * - getting the cpu usage,
16  * - getting the average cpu usage,
17  * - getting the RAM on device (total, used and available),
18  * - getting the partition size on device (total, used, and avalable),
19  * - getting the device temperature in platform storage,
20  * - getting the size of a file,
21  * - getting the size of a folder (directory),
22  * - Execute any Linux command through system call,
23  * - get the Linux version, and
24  * - get the Kernel version
25  * via the provided interfaces.
26  * The user shall use this interface to interact with the following modules:
27  * - Linux File System
28  *
29  * The user does not need to add any other headers to use this API.
30  *
31  * The following functionalities are provided by the API:
32  *
33  * NOTE: All functions were written only for the Linux Operating System.
34  *
35  * # Description of Execute functions:
36  * These functions allow the user to execute any Linux system command not done by any of the other functions.
37  *
38  * # Description of GetDate function:
39  * This function allows the user to retrieve the current clock date and time as a string
40  *
41  * # Description of GetCpuTemperature function:
42  * This function allows the user to retrieve the temperature of the processor in degrees Celsius.
43  *
44  * # Description of GetCpuUsage function:
45  * This function allows the user to retrieve the percentage of usage/utilization of the cpu.
46  *
47  * # Description of GetCpuUsageAverage function:
48  * This function allows the user to retrieve the average percentage of usage/utilization of
49  * multiple samples of cpu usage.
50  *
51  * # Description of GetTotalRam function:
52  * This function allows the user to retrieve the total amount of RAM in bytes on the system.
53  *
54  * # Description of GetUsedRam function:
55  * This function allows the user to retrieve the total amount of RAM in bytes used by the system.
56  *
57  * # Description of GetAvailableRam function:
58  * This function allows the user to retrieve the total amount of RAM in bytes not used by the system.
59  *
60  * # Description of GetMaxProcessRamUsed function:
61  * This function allows the user to retrieve the maximum amount of RAM in bytes used by the current
62  * process so far.
63  *
64  * # Description of GetPartitionTotalSize function:
65  * This function allows the user to retrieve the total number of bytes in the specified partition.
66  *
67  * # Description of GetUsedPartitionSpace function:
68  * This function allows the user to retrieve the total number of bytes used on the specified partition.
69  *
70  * # Description of GetAvailablePartitionSpace function:
71  * This function allows the user to retrieve the total number of bytes free (not used) on the specified partition.
72  *
73  * # Description of GetFileSize function:
74  * This function allows the user to retrieve the total number of bytes in the specified file.
75  *
76  * # Description of GetDirectorySize function:
77  * This function allows the user to retrieve the total number of bytes in the specified folder/directory.
78  *
79  * # Description of GetLinuxVersion function:
80  * This function allows the user to retrieve the value of the version of Linux used.
81  *
82  * # Description of GetKernelVersion function:
83  * This function allows the user to retrieve the value of the Linux Kernel version used.
84  *
