[in]cipher_typeInput for desired supported encryption method.
[in]certificate_version_numberVersion number of certificate.
[in]countryCountry of origin where certificate was generated.
[in]provinceProvince or State of origin, if applicable.
[in]cityCity of Origin, if applicable.
[in]organizationOrganization or Company name Certificate will be assigned to.
[in]commonAdditional information.
[out]cert_outEncoded cert request. Will return "error" if generation process fails.
Returns
