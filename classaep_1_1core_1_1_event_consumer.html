Detailed Description

AcpEvent consumer base class All event consumers should inherit from this class. Must implement and use EventShouldBeConsumed as the event filter Must implement and use ConsumeEvent as the event handler If the event consumer will be used concurrently, e.g., registered at two event queues, use lock() and unlock() function to make consumer thread-safe
