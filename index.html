edge-platform-sdk

Software Development Kit for Edge Platform.

Introduction

Checkout Edge Platform SDK from here if you want to develop applications on Edge Platform.

Language: C/C++

Getting Started

Install SDK

SDK can be downloaded using download_sdk.sh

Include

API headers are included in this folder and divded into the following categories.

  • core - Headers for framework core
  • ext - Headers for external libraries
  • platform_api - Headers for hardware abstraction layer
  • rpc - Headers for RPC

Lib

Prebuild libraries are included for application development.

Manifest

Sample system manifest files are provided.

Tools

Platform builder tools can be found in this folder.

Examples

Example applications can be found in this folder.

Building Applications

Checkout the example applications to see more details.

Generate System Image

Use the builder tool to create a system image that contains your application and needed platform services.

./tools/AcpSystemBuilder {root-folder} {manifest-file} {image-file}

For example, for base system image, create with following commands.

./tools/AcpSystemBuilder . manifest/sample_system_manifest.json system_image
