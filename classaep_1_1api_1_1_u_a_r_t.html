Detailed Description

This class is a child of SerialBus class. It must implement abstract methods of its parent class. UART stands for Universal Asynchronous Receiver-Transmitter. It is a asynchronous bidirectional protocol for low-speed communication. It is called asynchronous because there is no clock. The timing is predefined by the application and both receiver and transmitter must agree on the speed.
