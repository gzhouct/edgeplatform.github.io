12  * This header is the interface of the core LogStream API.
13  * The user will be able to perform the following operations to a log stream:
14  * - getting the total size and list of log files in a folder,
15  * - delete a log file,
16  * - compress a log file,
17  * - decompress a log file,
18  * - construct a LogStream class object,
19  * - set the log level,
20  * - set the log timing precision from microseconds to seconds,
21  * - set the log file name,
22  * - turn on logging to the console (in addition to the log file),
23  * - turn off logging to the console,
24  * - turn on log cycling,
25  * - turn off log cycling,
26  * - turn on log compression,
27  * - turn off log compression,
28  * - add a string message to queue to be logged,
29  * - construct a LogStreamLock class object
30  * - Stream object to the LogStreamLock class object
31  * - destroy a LogStreamLock class object.
32  * via the provided interfaces.
33  * The user shall use this interface to interact with the following modules:
34  * - All ACP classes can use this to log their data
35  *
36  * The user does not need to add any other headers to use this API.
37  * However each app (HUB, DAQ, OTA, etc.) may define their own header to log their own messages.
38  *
39  * The following functionalities are provided by the API:
40  *
41  *
42  * # Description of GetSizeAndListOfLogs function:
43  * This function allows the user to get the total size of logfiles in a folder and get a list of all the logfiles in that folder.
44  *
45  * # Description of DeleteSingleLog function:
46  * This function allows the user to delete the specified logfile.
47  *
48  * # Description of CompressLog function:
49  * This function allows the user to compress the specified logfile.
50  *
51  * # Description of DecompressLog function:
52  * This function allows the user to decompress the specified logfile.
53  *
54  * # Description of LogStream class:
55  * The log stream manages log messages from many sources.
56  * It is designed to be thread-safe so that a single global instance
57  * can accept messages from the entire program.
58  *
59  * # Description of LogStream constructor method:
60  * This method constructs a LogStream object.
61  *
62  * # Description of LogStream SetLogLevel method:
63  * This method sets the log-level for the LogStream stream.
64  *
65  * # Description of LogStream SetSecondsPrecision method:
66  * This method sets the log timing precision from microseconds to seconds,
67  *
68  * # Description of LogStream SetLogFile method:
69  * This method sets the name of the log-file
70  *
71  * # Description of LogStream SplitToConsoleOn method:
72  * This method enables copying the contents being written to a file to be shown on the console
73  *
74  * # Description of LogStream SplitToConsoleOff method:
75  * This method disables copying the contents being written to a file to be shown on the console
76  *
77  * # Description of LogStream CyclingOn method:
78  * This method enables log cycling
79  *
80  * # Description of LogStream CyclingOff method:
81  * This method disables log cycling
82  *
83  * # Description of LogStream CompressionOn method:
84  * This method turns on the log compression
85  *
86  * # Description of LogStream CompressionOff method:
87  * This method turns off the log compression
88  *
89  * # Description of LogStream QueueWrite method:
90  * This private method appends a new message to the log-file or log-file buffer.
91  *
92  * # Description of LogStream CycleLogs method:
93  * This private method handles the cycling of the log-files.
94  *
95  * # Description of LogStream CompressFile method:
96  * This private method compresses the specified log-file.
97  *
98  * # Description of LogStreamLock class:
99  * Temporary object to buffer input before sending to stream.
100  * This object will get instantiated when the user attempts to log
101  * a message. It buffers the input before streaming, providing a
102  * thread-safe way to send messages to the logger.
103  *
104  * # Description of LogStreamLock constructor method:
105  * This method constructs a LogStreamLock object.
106  *
107  * # Description of LogStreamLock operator<< method:
108  * This method streams an object to the stringstream stream
109  *
110  * # Description of LogStreamLock destructor method:
111  * This method destroys/frees a LogStreamLock object.
112  *
113  */
