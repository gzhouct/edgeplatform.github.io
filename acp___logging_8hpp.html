◆ ACP_LOG

#define ACP_LOG (   tag,
  level 
)    aep::core::LogMessage ((level), __FUNCTION__, (tag))

Macro used to capture the current function name for logging. The function name can be useful for DEBUG logs, but it can be a security risk if it ends up in production. Therefore, for any log level above DEBUG (including INFO, WARNING, and ERROR), will not show the function name.
