Detailed Description

This class is a child of SerialBus class. It must implement abstract methods of its parent class. SPI stands for Serial Peripheral Interface. It is a synchronous serial communication interface specification used for short-distance communication. SPI devices can communicate in half duplex (HD) or full duplex (FD) mode using a master-slave architecture with a single master.
