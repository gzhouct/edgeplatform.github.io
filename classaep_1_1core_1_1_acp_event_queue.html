Detailed Description

This class defines an event queue. AcpEvent queue is a priority queue. Events inserted into the event queue is ordered based on their priorities. A background work thread will pop the event from queue front. Once popped, subscribers of the event will be notified. Once all subscribers return, the event will be destroyed.

Note
queue will be blocked when subscriber handles the event.
